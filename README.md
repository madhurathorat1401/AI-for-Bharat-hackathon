StudentSaathi AI

StudentSaathi AI is a lightweight web application designed to support college students with academics, career preparation, and mental well-being in a single platform. The project focuses on simplicity, reliability, and student-centric design.

Features

Study Buddy for asking academic doubts from notes or PDFs
Placement Path generator with a realistic career roadmap
Wellness check with stress level detection and visual feedback
Burnout awareness through balanced scheduling logic

Tech Stack

Python
Flask
HTML
CSS
PyPDF
Rule-based logic (AI-inspired, no external APIs)

Project structure
student_saathi/

How to Run

Install dependencies
pip install flask pypdf

Run the application
python app.py

(Use python3 on macOS)
Open in a browser
http://127.0.0.1:5000

Notes

The project uses AI-inspired, rule-based logic to demonstrate intelligent behaviour without relying on paid APIs. The architecture is designed to be easily extendable with real AI models in the future.
